# Hybrid Hexapod Robot - "Hexy"

> A six-legged hybrid robot capable of moving using both walking and wheeled locomotion. Developed as a university engineering project focused on mechanical design, prototyping, and robotics integration.


> Basic movement

![fastMovement.gif](Hexy-20251124T231509Z-1-002/Hexy/hex%20v4.0/clips/fastMovement.gif)

## Project Overview

This project presents a hybrid hexapod robot that combines the stability of a six-legged platform with the efficiency of wheeled locomotion. The robot was designed and developed through multiple design iterations, gradually improving mechanical reliability, mobility, and manufacturability.

The project involved:

* Mechanical design and CAD modeling
* Rapid prototyping and 3D printing
* Electronics integration
* Locomotion mechanism development
* Basic robot control software

---

# Key Features

* Six-legged robot platform
* Hybrid wheel-leg locomotion
* Modular mechanical design
* Fully custom CAD models
* Multiple design iterations
* 3D printed structural components
